# Suggested Commands
- `npm install` (inside `extensions/ai-assistant`) – install dependencies.
- `npm run dev` – run `ray develop` for live reloading Raycast extension.
- `npm run build` – bundle extension via `ray build`.
- `npm run lint` / `npm run fix-lint` – run or auto-fix linting via Raycast CLI.
- `npm run publish` – publish extension to Raycast Store (requires auth).
- Raycast-specific: use Raycast app to test commands (Dictate Text, Dictate AI Prompt, Settings, etc.).
